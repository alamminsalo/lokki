# Milestone 53 — UI Console

_Purpose: Add a standalone CLI tool with a `ui` command that displays a TUI for browsing lokki flows, runs, and logs._

### Background

Users currently need to use AWS Console or multiple CLI commands (`show`, `logs`) to monitor flow executions. A unified UI console provides a better developer experience for browsing and debugging flows.

All deployed lokki resources are tagged with `lokki:managed=true` for discovery. Each run uses a dedicated CloudWatch log stream for complete log isolation.
